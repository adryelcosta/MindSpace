## 📦 Como Executar o Projeto

Para clonar e rodar esta aplicação localmente em seu computador, você precisará do [Git](https://git-scm.com) e do [Node.js](https://nodejs.org/) (ou outra ferramenta dependendo da sua tecnologia) instalados.

No seu terminal, execute os seguintes comandos:

```bash
# 1. Clone este repositório
$ git clone [https://github.com/adryelcosta/MindSpace.git](https://github.com/adryelcosta/MindSpace.git)

# 2. Acesse a pasta do projeto
$ cd MindSpace

# 3. Instale as dependências
$ npm install

# 4. Inicie a aplicação
$ npm start
